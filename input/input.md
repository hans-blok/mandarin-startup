# Thema-epic
Product Mandarin

# Probleemruimte
IT-professionals missen voorspelbaarheid, zichtbaarheid en verantwoordelijkheid bij impliciete tooling.

# Status-quo
Veel ad-hoc automatisering, weinig structurele borging.

# Verbetering
Een expliciet, governable en traceerbaar agent-ecosysteem.

# Aannames
- IT-professionals zijn bereid hun kennis expliciet te maken.
- Een agent-ecosysteem is daadwerkelijk beter voorspelbaar.
- De voordelen wegen op tegen de extra inspanning.

# Positionering
Koppeling met Product Mandarin, gericht op structurele kwaliteitsverbetering.
